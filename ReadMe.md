* Command
* Interpreter
* Iterator
* Mediator
* Memento
* Null Object
* Observer
* State
* Strategy
* Template
* Visitor

## Creational

* Abstract Factory
* Builder
* Factory
* Prototype
* Singleton

## Structural

* Adapter
* Bridge
* Composite
* Decorator
* Facade
* Filter
* Flyweight
* Proxy
